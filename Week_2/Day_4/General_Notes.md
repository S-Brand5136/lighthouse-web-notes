### Notes

#### JSON

- Json is built on two structures

  - A collection of name/value pairs.
  - An ordered list of values.

    ```JSON
      {
        "name": "New York City",
        "boroughs": [
          "Manhattan",
          "Queens",
          "Brooklyn",
          "The Bronx",
          "Staten Island"],
        "population": 8491079,
        "area_codes": [212, 347, 646, 718, 917, 929],
        "position": { "lat": 40.7127, "lng": -74.0059 }
      }
    ```

- Json syntax must be valid Javascript!
